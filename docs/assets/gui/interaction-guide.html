GUI 是可视化上下文入口

它不是替代 Codex 的控制台。它把本地文件、Obsidian、历史报告和 AI 对话归档整理成 Codex 接手包，让 Codex 接手时少读上下文。

默认只读：不删除、不移动、不重命名、不重写源文件

1. 先看上下文来源

右侧概览展示文件雷达、Obsidian 笔记数、历史报告和最近上下文素材。本地文件 / 目录目标可以先点检查本地目标。

2. 生成 Codex 接手包

写下要交给 Codex 的任务，系统会提取来源路径、相关原因、压缩摘要和边界。

3. 复制给 Codex

把 prompt 交给 Codex，它就能基于真实本地路径继续处理，不需要你重复解释。
