# Guidebook

本目录保存可直接给用户阅读或导入 NotebookLM 的完整教程资料包。

## 文件

- [knowledge-action-assistant-tutorial.pdf](knowledge-action-assistant-tutorial.pdf): 7 页 PDF 教程，覆盖从“今天先干什么”到闭环复盘的主流程。
- [slides/page-01.png](slides/page-01.png): 定位与四层结构。
- [slides/page-02.png](slides/page-02.png): 一天怎么开始。
- [slides/page-03.png](slides/page-03.png): 记录一个任务。
- [slides/page-04.png](slides/page-04.png): 内容放哪。
- [slides/page-05.png](slides/page-05.png): 文件雷达与 Obsidian 体检。
- [slides/page-06.png](slides/page-06.png): AI 上下文取用与闭环。
- [slides/page-07.png](slides/page-07.png): 一周使用节奏。

## 使用方式

1. 新用户先读 PDF，再打开 GUI 的“今天先干什么”。
2. 需要系统学习时，把 PDF 和 7 张分页图导入 NotebookLM。
3. 做演示或培训时，直接用 `slides/` 下的分页图逐页讲解。

## 安全边界

Guidebook 只解释使用流程，不要求用户把源文件交给工具自动移动、删除或重命名。助手默认只生成报告、建议、manifest 和明确的新笔记。
