本目录保存可直接阅读或导入 NotebookLM 的教程资料包。新版教程围绕本地知识整理助手的三个主入口组织：添加资料、搜索回顾、生成 AI 上下文包。

## 建议用法

1. 新用户先读 PDF，再打开 GUI。
2. 先跑 `添加资料` 写入一条 demo 记录。
3. 用 `搜索回顾` 查刚才写入的内容。
4. 用 `生成 AI 上下文包` 先预览候选来源，确认后生成可交给 AI 的上下文包。

Guidebook 只解释使用流程，不要求用户把源文件交给工具自动移动、删除或重命名。助手默认只生成报告、建议、manifest 和明确的新笔记。
