# Guidebook

本目录保存可直接阅读或导入 NotebookLM 的教程资料包。新版教程围绕本地知识整理助手的四个入口组织：整理资料、回顾知识、提取上下文、今日提醒。

## 建议用法

1. 新用户先读 PDF，再打开 GUI。
2. 先跑 `整理资料` 写入一条 demo 记录。
3. 用 `回顾知识` 查刚才写入的内容。
4. 用 `提取上下文` 生成 AI 上下文包。
5. 用 `今日提醒` 保持每天轻量，不要一上来处理全部 backlog。

Guidebook 只解释使用流程，不要求用户把源文件交给工具自动移动、删除或重命名。助手默认只生成报告、建议、manifest 和明确的新笔记。
